# ConectaMax

## Objetivo general

Desarrollar una aplicacion academica tipo CRM orientada al analisis y prediccion del abandono de clientes de una empresa ficticia de telecomunicaciones.

## Descripcion breve

ConectaMax sera una aplicacion en Python organizada por capas para separar configuracion, acceso a datos, logica de negocio, interfaz, utilidades, reportes, pruebas y documentacion. En esta etapa solo se prepara la estructura inicial del proyecto.

## Estructura de carpetas

```text
conectamax/
├── app.py
├── requirements.txt
├── README.md
├── .gitignore
├── config/
├── data/
├── database/
├── services/
├── views/
├── utils/
├── reports/
├── tests/
└── docs/
```

## Estado inicial del desarrollo

El proyecto se encuentra en fase inicial. Aun no se han implementado funcionalidades, interfaz, base de datos, analisis de datos ni modelo predictivo.
